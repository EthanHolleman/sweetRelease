# sweetRelease

A python script to release all your stuck SLURM jobs from "requeue held"

`sweetRelease --help`
```
usage: sweetRelease [-h] [--quiet QUIET] user

Release all SLURM held jobs for a user (yourself).

positional arguments:
  user           Your username

optional arguments:
  -h, --help     show this help message and exit
  --quiet QUIET  Do not produce output.
```
